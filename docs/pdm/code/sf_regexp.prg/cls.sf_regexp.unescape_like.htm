UnEscape_Like Method
Class: SF_RegExp
Description:Interface to the DotNet UnEscape method, with special operation to keep LIKE function
Converts any escaped characters in the input string, but keep the wildcard funtion of Like .
RetVal 
=
Object.UnEscape_Like
(tvValue)
Parameter
tvValue
The string (usually a Pattern) to unescape
Type Character
Direction Input
Return value String
The string unescaped.
Remarks

Just like RegExp UnEscape, but alters ".*" and "." into "*?"

Note the restrictions in UnEscape.

Source Code 
  LPARAMETERS;
   tvValue

  LOCAL;
   lvReturn AS STRING

  lvReturn = ""+STRTRAN(STRTRAN(STRTRAN(STRTRAN(m.tvValue,".*","\*"),"\.",0h00),".","\?"),0h00,"\.")
  RETURN THIS.roRegExp.UnEscape(m.lvReturn)
