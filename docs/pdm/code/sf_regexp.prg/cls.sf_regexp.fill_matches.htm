Fill_Matches Method
Class: SF_RegExp
Description:Rebuild the Matches object returned by DotNet to VFP Collections object
Converts a DotNet Matches object into a VFP Collection.
RetVal 
=
Object.Fill_Matches
(toMatches [,tlAutoExpandGroups [,tlAutoExpandCaptures]])
Parameter
toMatches
Matches collection
Type Object
Direction Input
tlAutoExpandGroups
Expand Groups in Matches to VFP Collections.
Type Boolean
Direction Input
Optional  

While expanding a Match object of toMatches, expand the Groups into a VFP Collection.

If the parameter is false, the DotNet object will be included.

tlAutoExpandCaptures
Expand Captures in Matches and Groups to VFP Collections.
Type Boolean
Direction Input
Optional  

While expanding the first Match object of toMatches, expand the Groups object into a VFP Collection.

If tlAutoExpandGroups is true, the expanded Groups will hold expanded Groups object as a VFP Collection.

If the parameter is false, the DotNet object will be included.

Return value Object
The VFP collection.
Remarks
This method is called by the Matches method, if ReturnFoxObjects is set. tlAutoExpandGroups will follow AutoExpandGroups, tlAutoExpandCaptures will follow AutoExpandCaptures.
See also
AutoExpandCaptures | AutoExpandGroups | Matches | ReturnFoxObjects
Source Code 
  LPARAMETERS;
   toMatches,;
   tlAutoExpandGroups,;
   tlAutoExpandCaptures

  LOCAL;
   lnMatch   AS INTEGER,;
   loMatches AS COLLECTION

  loMatches = CREATEOBJECT("Collection")
  FOR lnMatch = 0 TO m.toMatches.COUNT-1
*   loMatches.ADD(THIS.Fill_Match(toMatches.ITEM(m.lnMatch),m.tlAutoExpandGroups,m.tlAutoExpandCaptures,m.lnMatch=0))
   loMatches.ADD(THIS.Fill_Match(toMatches.ITEM(m.lnMatch),m.tlAutoExpandGroups,m.tlAutoExpandCaptures,.T.))
  ENDFOR &&lnMatch

  RETURN m.loMatches
