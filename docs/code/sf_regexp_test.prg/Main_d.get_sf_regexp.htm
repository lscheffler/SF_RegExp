Get_SF_RegExp Procedure
Returns a SF_RegExp object for testing purposes.
RetVal 
=
Get_SF_RegExp
()
Return value Object
SF_RegExp object.
Source Code 
 LOCAL;
  lcOldPath AS STRING,;
  lcPath    AS STRING,;
  loRegExp  AS OBJECT

 lcPath    = SYS(16)
 lcPath    = RIGHT(m.lcPath,LEN(m.lcPath)-AT(" ",m.lcPath,2))
 lcPath    = JUSTPATH(m.lcPath)
 lcOldPath = FULLPATH("","")

 CD (m.lcPath)

 loRegExp = SF_RegExp()

 CD (m.lcOldPath)
 RETURN m.loRegExp
