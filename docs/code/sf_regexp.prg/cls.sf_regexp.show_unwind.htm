Show_Unwind Method
Class: SF_RegExp
Description:Expand to a Matches or Match object for to show all data in the object
Unwinds a whole Matches or Match DotNet or VFP object returned.
RetVal 
=
Object.Show_Unwind
()
Return value String
A text representation of a Matches / Match object.
Remarks

This shows how to get all properties returned by the Matches or Match.

The idea is to get all return of Match / Matches in visual style for debuging.

If a VFP object is used, see ReturnFoxObjects, Groups and Captures will be visible in the output only if they are included into the object if AutoExpandGroups and AutoExpandCaptures are set.

See also
AutoExpandCaptures | AutoExpandGroups | Matches | ReturnFoxObjects
Source Code 
  LPARAMETERS;
   toMatches

  LOCAL;
   lcReturn  AS STRING,;
   llMatches AS INTEGER

*  SET STEP ON

  llMatches = .T.
  TRY
    =m.toMatches.COUNT
   CATCH
    llMatches = .F.
  ENDTRY

  IF m.llMatches THEN
*Matches
   lcReturn = "Matches "
   IF PEMSTATUS(m.toMatches,"tag",5) THEN
*VFP object
    lcReturn =  "Expanding VFP " + m.lcReturn + 0h0D0A + THIS.Unwind_Matches_V(m.toMatches)
   ELSE  &&PEMSTATUS(m.toMatches,"tag",5)
*DotNet object
    lcReturn =  "Expanding DotNet " + m.lcReturn + "Matches" + 0h0D0A + THIS.Unwind_Matches(m.toMatches)
   ENDIF &&PEMSTATUS(m.toMatches,"tag",5)   ELSE  &&m.llMatches

  ELSE  &&m.llMatches
*Matches 
   lcReturn = "Match "
   IF PEMSTATUS(m.toMatches,"value",5) THEN
*VFP object
    lcReturn =  "Expanding VFP " + m.lcReturn + 0h0D0A + THIS.Unwind_Match_V(m.toMatches)
   ELSE  &&PEMSTATUS(m.toMatches,"value",5)
*DotNet object
    lcReturn =  "Expanding DotNet " + m.lcReturn + 0h0D0A + THIS.Unwind_Match(m.toMatches)
   ENDIF &&PEMSTATUS(m.toMatches,"value",5)
  ENDIF &&m.llMatches

  RETURN m.lcReturn
