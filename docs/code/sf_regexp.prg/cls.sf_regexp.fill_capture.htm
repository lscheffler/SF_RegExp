Fill_Capture Method
Class: SF_RegExp
Description:Rebuild the Capture object returned by DotNet to VFP Empty object
Converts a DotNet Capture object into a VFP Empty object.
RetVal 
=
Object.Fill_Capture
(toCapture [,tlWithName])
Parameter
toCapture
Capture object
Type Object
Direction Input
tlWithName
Expand Name and Success property of the Capture object included.
Type Boolean
Direction Input
Optional  

This is only true for the first Capture of a Group (or Match), else it would fail.

This is set by the calling : Fill_Group. method.

Return value Object
The VFP Empty object.
Source Code 
  LPARAMETERS;
   toCapture,;
   tlWithName

  LOCAL;
   loCapture AS EMPTY

  loCapture = CREATEOBJECT("Empty")
  ADDPROPERTY(m.loCapture,"Value", THIS.Get_Value (m.toCapture))
  ADDPROPERTY(m.loCapture,"Index", THIS.Get_Index (m.toCapture))
  ADDPROPERTY(m.loCapture,"Length",THIS.Get_Length(m.toCapture))
  IF m.tlWithName THEN
   ADDPROPERTY(m.loCapture,"Name",   THIS.Get_Name   (m.toCapture))
   ADDPROPERTY(m.loCapture,"Success",THIS.Get_Success(m.toCapture))
  ENDIF &&m.tlWithName

  RETURN m.loCapture
