Get_Groups Method
Class: SF_RegExp
Description:Get the Groups property from DotNet Match objects
Source Code 
  LPARAMETERS;
   loMatch

  LOCAL;
   loBridge AS OBJECT,;
   lvReturn AS VARIANT

  loBridge = GetwwDotnetBridge()
  lvReturn = loBridge.GetProperty(m.loMatch,"Groups")
  RETURN m.lvReturn
