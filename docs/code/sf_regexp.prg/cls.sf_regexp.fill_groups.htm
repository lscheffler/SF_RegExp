Fill_Groups Method
Class: SF_RegExp
Description:Rebuild the Groups object returned by DotNet to VFP Collections object
Converts a DotNet Groups object into a VFP Collection.
RetVal 
=
Object.Fill_Groups
(toGroups [,tlAutoExpandCaptures])
Parameter
toGroups
Groups collection
Type Object
Direction Input
tlAutoExpandCaptures
Expand Captures in Group objects to VFP Collections.
Type Boolean
Direction Input
Optional  

If the parameter is false, the DotNet object will be included.

Return value Object
The VFP collection.
Source Code 
  LPARAMETERS;
   toGroups,;
   tlAutoExpandCaptures

  LOCAL;
   lnGroup  AS INTEGER,;
   loGroups AS COLLECTION

  loGroups = CREATEOBJECT("Collection")
  FOR lnGroup = 0 TO m.toGroups.COUNT-1
   loGroups.ADD(THIS.Fill_Group(toGroups.ITEM(m.lnGroup),m.tlAutoExpandCaptures))
  ENDFOR &&lnGroup

  RETURN m.loGroups
