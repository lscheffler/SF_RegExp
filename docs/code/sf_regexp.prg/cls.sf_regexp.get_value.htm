Get_Value Method
Class: SF_RegExp
Description:Get the Value property from DotNet Match, Group, Capture objects
Returns the Value property from an object used by DotNet RegExp.
RetVal 
=
Object.Get_Value
(loMatch)
Parameter
loMatch
The object where Value is to be read from.
Type Object
Direction Input
This works for Match, Group, Capture.
Return value String
The value, depending on the input.
Remarks

For a larger example how to use this, see Test_Unwind in example program.

See also
example program
Source Code 
  LPARAMETERS;
   loMatch

  LOCAL;
   loBridge AS OBJECT,;
   lvReturn AS VARIANT

  loBridge = GetwwDotnetBridge()
  lvReturn = loBridge.GetProperty(m.loMatch,"Value")

  RETURN m.lvReturn
